Содержание курса:



Веб-разработка сейчас - одно из самых востребованных направлений разработки ПО. Сайт нужен каждой компании, а многим компаниям даже не один, поэтому данное направление очень перспективно. В курсе веб-разработки мы познакомим вас с клиентской веб-разработкой, которая включает в себя верстку - обучение HTML и CSS, и написание кода на JavaScript. На курсе JavaScript будут рассмотрены синтаксис, типы данных, работа с DOM и событиями, стандартная библиотека языка. В курсе JavaScript также рассматривается большое количество библиотек и средств разработки - jQuery, jQuery UI, Underscore, Vue.js, Bootstrap, npm, Webpack, Less/Sass, Babel. Также обзорно рассматриваем Grunt, RequireJS, TypeScript. В качестве серверной части используем Express.js. Обучение веб-разработке закрепляется на практике. Вы научитесь создавать клиентскую часть сайта и получите большой кругозор по клиентским технологиям.
